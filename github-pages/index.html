🧠 Anthropic Advisor Tool Beta

A server-side Anthropic feature that pairs a fast executor model (Sonnet or Haiku) with a smarter advisor model (Opus). Midway through generation, the executor calls the advisor — Anthropic runs a separate Opus sub-inference that reads the full conversation and returns strategic guidance. Everything happens inside one API call with no extra round trips.

Connect in 2 Lines

Python 
# Standard chat
client = OpenAI(
    api_key="sk-your-plaudeapi-token",
    base_url="https://plaudeapi.com/v1",
)

# Force Batch tier (50% off) on any model
response = client.chat.completions.create(
    model="claude-sonnet-4-6",
    messages=[{"role": "user", "content": long_document}],
    extra_headers={"X-Tier": "batch"},
)

# Enable Advisor Tool — middleware injects Opus advisor automatically
# Sonnet executor consults Opus mid-generation for strategic guidance
response = client.chat.completions.create(
    model="claude-sonnet-4-6",
    messages=[{"role": "user", "content": complex_coding_task}],
    extra_headers={"X-Ploud-Advisor": "true"},
)
